COVID Analysis using SQL & Tableau

Purpose: Demonstrate skills by analyzing publicly available global COVID-19 data. This data includes population, cases, deaths, vaccinations, and hospital/ICU admissions by date.

SQL skills used: Joins, CTE, Temporary tables, Aggregate Functions, Windowed aggregate function, Creating Views, Converting Data Types

  • Download data in csv format from https://ourworldindata.org/covid-deaths
  • Import into a new PostgreSQL database, called COVID with 2 tables
    • Deaths data
    • Vaccinations data
  • Check/correct column data types
  • Convert date text string to separate year, month, day numeric columns

Analysis

Total Cases vs. Total Deaths in the US (likelihood of dying when contracting COVID)

SELECT location, date, total_cases, total_deaths
   ,CAST(total_deaths AS REAL)/ total_cases * 100 AS death_percentage FROM deaths
WHERE location = ‘United States’
ORDER BY location, date

The graph below shows how the death percentage decreased over time as treatment options improved and successive variants became less lethal (even if they were more contagious).

The graph below shows the evolution over time of total COVID cases. Significant increases are noted in the winter of 2020/2021 and 2021/2022.

Countries with Highest Per Capita Infection Rate

Note: The infection percentage is probably not accurate due to people contracting COVID multiple times.

SELECT location, population
   ,MAX(total_cases) as MaxCases
   ,MAX(total_cases * 1.0 / population) * 100.0 as PercentPopulationInfected
FROM deaths
WHERE continent <> ‘’ AND total_cases IS NOT NULL
GROUP BY location, population
ORDER BY PercentPopulationInfected DESC

Countries with Highest Number of Deaths

SELECT location, MAX(total_deaths) AS TotalDeathCount
FROM deaths
WHERE continent <> ‘’ AND total_deaths IS NOT NULL
GROUP BY location
ORDER BY TotalDeathCount DESC

Total Deaths Globally and by Continent

SELECT location, MAX(total_deaths) as TotalDeathCount
FROM deaths
WHERE continent = ‘’
GROUP BY location

Global Number of Deaths as of 2023 Apr 08

SELECT SUM(new_cases) AS TotalCases
   ,SUM(new_deaths) AS TotalDeaths
   ,SUM(new_deaths * 1.0) / NULLIF(SUM(new_cases),0) * 100 AS DeathPercentage
FROM deaths
WHERE continent IS NOT NULL

Join deaths and vaccinations tables

Running Vaccination Totals by Country

Note: The vaccination percentage is probably not accurate due to booster vaccinations.

SELECT d.continent, d.location, d.date, d.population    ,v.new_people_vaccinated_smoothed
   ,SUM(v.new_people_vaccinated_smoothed)
      OVER (PARTITION BY d.location ORDER BY d.location, d.date) AS VaccinationSum
FROM deaths d
JOIN vaccinations v
ON d.location = v.location
AND d.date = v.date
WHERE d.continent IS NOT NULL AND d.continent <> ‘’
ORDER BY 2, 3

Running Vaccination Percentage by Country using CTE

WITH PopvsVaccination (continent, location, date, population
   , new_people_vaccinated_smoothed, VaccinationSum) AS
(
SELECT d.continent, d.location, d.date, d.population, v.new_people_vaccinated_smoothed
   ,SUM(v.new_people_vaccinated_smoothed)
      OVER (PARTITION BY d.location ORDER BY d.location, d.date) AS VaccinationSum FROM deaths d
JOIN vaccinations v
ON d.location = v.location
AND d.date = v.date
WHERE d.continent IS NOT NULL AND d.continent <> ‘’
)

SELECT *, VaccinationSum * 1.0 / population * 100 AS PercentVaccinated
FROM PopvsVaccination

Running Vaccination Percentage by Country using Temporary Table

– Drop table if it exists
DROP TABLE IF EXISTS PercentPopulationVaccinated;

– Create the table and set data types
CREATE TEMPORARY TABLE PercentPopulationVaccinated
(
continent varchar(50),
location varchar(50),
date date,
population int8,
new_people_vaccinated_smoothed int4,
VaccinationSum int8
);

– Insert data into table from SELECT statement
INSERT INTO PercentPopulationVaccinated
SELECT d.continent, d.location, d.date
   ,d.population, v.new_people_vaccinated_smoothed
   ,SUM(v.new_people_vaccinated_smoothed) OVER (PARTITION BY d.location ORDER BY d.location, d.date) AS VaccinationSum
FROM deaths d
JOIN vaccinations v
   ON d.location = v.location
   AND d.date = v.date
WHERE d.continent IS NOT NULL AND d.continent <> ‘’;

– Show data from temp table
SELECT *, (VaccinationSum * 1.0 / population) * 100 AS PercentVaccinated
FROM PercentPopulationVaccinated;

Create View of Vaccination Sum to Store Data for Visualizations

DROP VIEW IF EXISTS vPercentPopulationVaccinated;

– Create view CREATE VIEW vPercentPopulationVaccinated AS
SELECT d.continent, d.location, d.date, d.population,
   v.new_people_vaccinated_smoothed
   ,SUM(v.new_people_vaccinated_smoothed)
      OVER (PARTITION BY d.location ORDER BY d.location, d.date) AS VaccinationSum
FROM deaths d
JOIN vaccinations v
   ON d.location = v.location
   AND d.date = v.date
WHERE d.continent IS NOT NULL AND d.continent <> ‘’
ORDER BY 2, 3;

– Show view
SELECT *
FROM vPercentPopulationVaccinated;
